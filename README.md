# schopen
Landing-Page für meine Shopping-Feed-App Schopen

## Features

- TikTok-ähnlicher, unendlicher Produktfeed mit Hero-Intro
- AWIN-kompatible Trackinglinks &amp; Affiliate-Hinweis
- Produktkarten mit Preisen, Rabatt, Tags und Trust-Badges
- Clientseitige Filter für Beauty, Tech, Fashion &amp; Home
- Offline-freundliches Lazy-Loading mit Retry &amp; Pull-to-Refresh

## Änderungen auf GitHub pushen
1. Melde dich mit `gh auth login` oder per klassischem SSH-Setup bei GitHub an.
2. Verknüpfe das Repository einmalig mit deinem Remote: `git remote add origin git@github.com:Ronaldocodingxx/schopen.git`
3. Sende den aktuellen Stand hoch: `git push -u origin work`
4. Für weitere Updates reicht künftig `git push` nach jedem lokalen Commit.
